# Music-can-weep

A discord music bot made by MCW

## Main features : 
  - play music from youtube, soundcloud url
  - play music from search keyword
  - pausing, resuming
  - modifying volume, pitch, tempo
  - queuing tracks, skiping
  - autoplaying
  - download
  
  
## Unstable features :
  - lyrics syncing
  - seeking
